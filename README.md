# URL Shortener (AWS + Node.js)

A simple URL shortener built using Node.js, Express, DynamoDB, and NanoID.

## 🚀 Features

- Shorten long URLs
- Redirect using short links
- Track click counts
- AWS DynamoDB integration
- Unique IDs using NanoID

## 🛠 Tech Stack

- Node.js
- Express.js
- AWS DynamoDB
- NanoID

npm install
